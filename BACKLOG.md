# ⏳ Бэклог разработки SigmaTrack

### 📊 Модуль «Обзор & Аналитика» (Dashboard)
- [ ] Главный экран проекта со сводной аналитикой (метрики, графики сгорания задач).
- [ ] Лента активности команды (кто что создал/закрыл).

### 📋 Модуль «Канбан-доска» (Kanban Board)
- [ ] Интеграция Drag-and-Drop для перемещения карточек задач между статусами.
- [ ] Кастомизация колонок под нужды проекта.

### 🏃‍♂️ Модуль «Спринты» (Sprints)
- [ ] Создание, планирование, запуск и закрытие итераций.
- [ ] Интеграция со Story Points (подсчет velocity команды).

### 👤 Пользователи и Профили (Улучшение)
- [ ] Расширенные настройки профиля (аватары, личные данные, навыки).
- [ ] Публичный просмотр профиля другого участника команды.
- [ ] Полнотекстовый поиск сотрудников для приглашения в проекты.

### 📎 Инфраструктура и Хранилище
- [] File Storage (Подсистема хранения картинок и документов для задач, комментариев и аватаров).
